# Air Power Data

This is a collection of data files and accompanying documents for JD Webster's “Air Power” rules for modern air-combat gaming.

## Aircraft Data Cards

I produce annotated aircraft data cards (ADCs) in PDF for version 1, 2, and 3 rules.

All ADCs in a single file are here:

- [version 1](https://github.com/alanwatsonforster/air-power-data/blob/main/aircraftdatacards/collections/version-1/all.pdf)
- [version 2](https://github.com/alanwatsonforster/air-power-data/blob/main/aircraftdatacards/collections/version-3/all.pdf)
- [version 3](https://github.com/alanwatsonforster/air-power-data/blob/main/aircraftdatacards/collections/version-3/all.pdf)
